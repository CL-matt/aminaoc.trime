# Aminaoc-Rime (阿米诺斯语输入法)

基于 Trime (同文输入法) 的阿米诺斯语 (Aminaoc) 专用输入方案。

## 特色
- **从右向左书写手感**：新字母自动推挤旧字母向右，模拟 RTL 书写体验。
- **自动反转逻辑**：输入拼音，自动输出对应的阿米诺斯语单词。
- **中文辅助提示**：候选框保留中文含义，方便拼写确认。

## 安装说明
1. **安装内核**：下载并安装 [Trime 同文输入法](https://github.com/osfans/trime/releases)。
2. **下载配置**：在本项目 [Releases](https://github.com/CL-matt/aminaoc.trime/releases) 处下载最新的 `aminaoc.trime.zip`。
3. **部署文件**：
   - 将压缩包内的 `luna_pinyin.custom.yaml` 和 `rime.lua` 解压到手机存储目录：`/storage/emulated/0/rime` (或 `/sdcard/rime`)。
   - 如图![Screenshot_20260321_193153_com_hihonor_filemanager_StorageActivity](https://github.com/user-attachments/assets/d345fc89-a320-464e-8c54-0ee2eda88264)

4. **激活方案**：
   - 打开同文输入法，点击 **“部署”**。
