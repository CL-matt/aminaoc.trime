# aminaoc.trime
基于trime的阿米诺斯语输入法（使用RTL）
# 使用说明
1. 下载trime 同文输入法
2. 下载 release 最新文件
3. 解压 aminaoc.trime.zip 到 storage/emulated/0/rime （此处为手机储存默认位置）
